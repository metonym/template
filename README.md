# vite-plugin-name

> Vite plugin

## Installation

```bash
# Yarn
yarn add -D vite-plugin-name

# NPM
npm i -D vite-plugin-name

# pnpm
pnpm i -D vite-plugin-name
```

## Usage

```js
// vite.config.js
import { defineConfig } from "vite";
import { plugin } from "vite-plugin-name";

export default defineConfig({
  plugins: [plugin()],
});
```

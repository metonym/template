# svite

> Template for building Svelte apps with [svite](https://github.com/dominikg/svite).

## Quick Start

```sh
npx degit metonym/template#svite svelte-app
cd svelte-app
yarn
```

## Available Scripts

### `yarn start`

Start the app in development mode.

### `yarn build`

Build the app for production.

# astro-site

> Template for building websites using Astro and Svelte.

## Quick Start

```sh
npx degit metonym/template#astro-site astro-site
cd astro-site
npm i
```

## Available Scripts

### `npm run dev`

Start the project in development mode.

### `npm run build`

Build the project for production.

### `npm run preview`

Preview the production build.
